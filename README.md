# GerenciaBanco 🏦

Aplicação de gerenciamento bancário desenvolvida em Java, utilizando os conceitos do paradigma de Orientação a Objetos.

## 📋 Sobre o projeto

Projeto acadêmico desenvolvido durante a disciplina de Linguagem Orientada a Objetos. A aplicação simula operações básicas de um sistema bancário, permitindo ao usuário cadastrar seus dados e realizar operações financeiras.

## ⚙️ Funcionalidades

- Cadastro de nome, sobrenome e CPF
- Consulta de saldo
- Depósito
- Saque
- Encerramento com mensagem de despedida

## 🛠️ Tecnologias utilizadas

- Java 21 (JDK LTS)
- NetBeans IDE
- Maven

## 📚 Conceitos aplicados

- Classes e Objetos
- Atributos e Métodos
- Instanciação de objetos
- Estruturas de controle (do...while e switch...case)
- Pacote Scanner para leitura de dados

## 🎓 Informações acadêmicas

- Disciplina: Linguagem Orientada a Objetos
- Curso: Sistemas para Internet
